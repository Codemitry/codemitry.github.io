Bio

In school, at the lessons of Informatics, teacher introduced me to Pascal. I wondered that it's possible to create program in fact by implementing my own knowledge.

When I was 15, I wanted to create programs, the result of which would be visible. I've chosen Delphi to achieve my aims. These programs were various calculators, mini-games and plain pair of clients and server thanks to which it's possible to communicate with each other. I understood that Delphi is ancient technology and craved changes.

At age 16, I became acquainted with Python. Just basics, principles of Object-oriented programming, tkinter, numpy, pandas. It's funny fact, I found out that I could create app for my Android-device using library called kivy for Python. Sure, I tried. The assembly process was terrible. it was done with buildozer library under Linux. Build process took half an hour.
